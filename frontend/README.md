- React 19
- TypeScript
- Tailwind CSS v4 via `@tailwindcss/vite`

The implementation plan lives at
`docs/plans/2026-05-25-runs-frontend-implementation.md`.

Before implementing UI, read `../DESIGN.md` and follow it strictly.
